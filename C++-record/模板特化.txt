模板特化过后，遇到对应类型，则会进入特化部分运行

例：

template <class T>
T Max(T a,T b){
    return a < b ? b : a;
}

//特化模板（部分特化）
template<class T>
T Max(T a,int b){
    return  a < b ? b : a;
}


调用的时候如果出现 Max(1.0,2)之类的则会自动匹配到特化的模板函数。
如果所有类型都被定义，则为完全特化。只定义一部分则为部分特化。
注意：特化模板必须写在通用模板之后。编译器会首先找到通用模板再进行特化。



在类中的属性的类型，如果使用类型的时候要用 typename 指明这是类型
