# VIM

> 按下ESC，进入Normal模式过后，所有的键都是功能键

> 设置高亮 :set hls

> 设置行号 :set number，该操作只会设置当前文档，如果需要每次打开文档都带有行号则需要设置vimrc文件。



## setp 01
> i
>> insert模式 ，按下ESC即回到Normal模式

> x
>> 删除当前光标所在的一个字符

> :wq
>> :w为存盘, :w后可跟文件名

>> :q为退出

> dd
>> 删除当前行，并拷贝到剪切板。

> p/P
>> 粘贴，p为在当前位置后粘贴，P为在当前位置前粘贴

> hjkl（键盘上依次向右）
>> h为向左，j为下一行，k为上一行，l为向右

> :help [command]
>> 显示帮助命令

## step 02
> a/A
>> 在光标后插入

>> 在行尾插入，相当于$a

> o/O
>> o为在当前行后插入一个新行

>> O为在当前行前插入一个新行

> cw
>> 替换从光标所在位置到一个单词结尾的字符

> nG
>> G为大写，将光标定位到第n行，效果与:n相同

> gg
>> 到第一行，同:1

> G
>> 到最后一行

> w/W 
>> w: 到下一个单词开头（将单词视为由字母数字下划线组成）， word_1 xxx

>> W: 同w，但是将单词视为被空字符分开的部。

> e/E
>> e: 到下一个单词结尾

>> E: E与e同w与W

> 0（数字）
>> 光标定位到行头

> $
>> 光标定位到本行行尾

> ^ 
>> 光标定位到本行第一个非空字符，空字符即空格，tab换行回车等

> g_
>> 光标定位到本行最后一个不是空字符的位置

> /pattern、?pattern 和 n/N
>> /pattern正向搜索pattern字符串（如果搜索到多个匹配项，按n键到下一个）
 
>> ?pattern反向搜索pattern字符串

>> n重复/pattern或?pattern

>> N回退/pattern或?pattern

> f/Ftooot 与 ;/,
>> find tooot，下一个，即将光标移动到当前位置后的第一个xxxx前

>> FIND tooot，上一个，即将光标移动到当前位置前的第一个xxxx前

>> ;可以重复上一次使用f命令查找的字符

>> ,可以反向查找，与;相反

> :s/target/replacement与 &
>> 执行替换

>> &为重复上一次的替换操作

> t/Ttooot
>> to tooot，下一个，即将光标移动到当前位置后的第一个xxxx的前一个字符前

>> TO tooot，上一个

> yy
>> 拷贝当前行，相当于 ddP

> u
>> 撤销，即undo

> Ctrl + u
>> 重做，即redo


## 区域选择
> visual模式下

> ＜action＞a＜object＞和＜action＞i＜object＞ 
>> action为任何命令，如 d（删除），y（拷贝），v（可视模式）

>> object可以是: w（一个单词），W（一个以空格为分隔符的单词），s（一个句子），p（一个段落），亦可以是一个字符："、'、)、]、}等

>> 假设有一个字符串是 (map (+) ("foo"))，且此时光标位于第一个o
- vi" ：选择foo
- va" ：选择"foo"
- vi) ：选择"foo"
- va) ：选择("foo")
- v2i) ：选择 map (+) ("foo")
- v2a) ：选择 (map (+) ("foo"))

> 块操作 Ctrl + v
>> e.g.:   0＜Ctrl+v＞＜Ctrl+d＞I--[ESC]
- 0: 到行头
- ＜Ctrl+v＞ 开始块操作
- ＜Ctrl+d＞ 移动选择区域，当然可以使用其它方式移动光标，如hjkl或者是%等
- I--: 插入"--"字符
- [ESC]: 使前面的操作生效
>> J: 把所有可视化中选中的行连接成一行

>> <或>: 左右缩进

>> =: 自动缩进


## 文件操作
> VIM可以同时打开多个文件

> :e＜file＞
>> 打开一个文件

> :w
>> 存盘

> :saveas ＜file＞
>> 另存为 file 文件

> :x, ZZ, :wq
>> 保存并退出， :x表示仅需要时保存

> :q!
>> 强制退出，不保存

> :qa!
>> 强制退出所有正在编辑的文件

> :bn/:bp
>> 切换上一个或下一个文件。


## 重复
> .
>> 重复上一次命令的修改，即上一次的命令要涉及到修改文档

> n＜Command＞
>> e.g.:
2dd : 删除两行
3p: 粘贴文本三次
100ixxxx [ESC]: 会将xxxx重复100次，并推出insert模式到 Normal模式

## 匹配
> %
>> （需先将鼠标移到括号上）匹配括号，并在括号间移动

> */#
>> 匹配当前单词

>> *: 下一个

>> \#: 上一个

> ＜satrt_position＞ ＜command＞ ＜end_position＞
>> e.g.: 0y$
>>> 0: 行头

>>> y: 拷贝

>>> $: 行尾

> ye
>> 拷贝当前位置到本单词最后一个字符

> y2/foo
>> 拷贝两个"foo"之间的字符串

> 并不是需要y才能拷贝，如下命令同样会被拷贝：
- d（删除）
- v（可视化的选择）
- gU（变大写）
- gu（变小写）


## 命令搭配
> 复合命令
- C : c$  // 剪切当前到本行末尾
- s : cl  // 剪切当前到下一个字符，即剪切一个字符
- S : ^c  // c为剪切，剪切位置紧跟着c，如cl剪切下一个字符，c$剪切本行
- I : ^i  // 在行首准备插入
- A : $a  // 在行尾准备插入
- o : A＜CR＞  // 当前行后插入一行
- O : ko  // 当前行前插入一行

## 自动提示
在insert模式，输入一个词的开头，然后按下＜Ctrl+p＞或者＜Ctrl+n＞就能自动补齐。


## 宏录制
> :qx 将操作记录在寄存器x

> @x: 重复录制的操作，即取寄存器x中的操作

> @@: 取最新录制的操作

>> e.g.: 在一个只有一行且只有"1"的文本中、
- qaYp＜Ctrl+a＞q
- qa: 开始记录操作
- Yp: 复制行
- ＜Ctrl+a＞: 增加1
- q: 停止复制
          
          @a: 在1下面写下2
          @@: 在2下面写下3
          100@@: 创建新的100行，每一行数据递增，直到103
          

## 分屏
> :split和:vsplit
>> :split创建水平分屏，:vsplit创建垂直分屏

>> ＜Ctrl+w＞＜dir＞
>>> dir为方向，可以是hjkl或者是键盘的方向键，它们用来切换分屏

>>> ＜Ctrl+w＞_或者＜Ctrl+w＞|
>>>> ＜Ctrl+w＞_ : 最大化尺寸

>>>> ＜Ctrl+w＞| : 用于垂直分屏

>>>> ＜Ctrl+w＞+或者＜Ctrl+w＞- : 增加尺寸
