

# Vue 源码

Vue 源码中主要涉及到以下部分比较重要：
- 响应式原理
- 模板编译
- diff 原理
- scheduler

- 带有 `g` 标志的正则表达式，在执行一次 exec 之后，会保留 lastIndex，下次匹配会从 lastIndex 开始查找，不带 `g` 标志的则 `lastIndex` 始终为 0

```js
var text = '\n        {{a + b}}\n        ';
var reg = /\{\{((?:.|\r?\n)+?)\}\}/g;
reg.exec(text); // ['{{a + b}}', 'a + b', index: 9, input: '\n        {{a + b}}\n        ', groups: undefined]
reg.exec(text); // null
// 重复
reg.exec(text); // ['{{a + b}}', 'a + b', index: 9, input: '\n        {{a + b}}\n        ', groups: undefined]
reg.exec(text); // null
```
