3、哪些元素的 offsetParent 为 null，哪些元素的 offsetParent 为 ＜body＞

为 null 的情况

  • 该元素为 ＜body＞
  • 元素不具有布局（如display:none）
  • 元素不存在于 DOM 中
  • 元素 position 为 fixed

为 body 的情况

  • 如果元素不是任何一个定位元素的后代，也不满足任何返回为 null 的条件，那么它的 offsetParent 为 ＜body＞
